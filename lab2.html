Tool Name: Android Studio

Platform: Android Devices

Application Development Type:

a: Native Application

b: Only one software needed to install and use

c: No redundancy

Website: developer.android.com

Developers: Google and JetBrains

First Version Release Date: May 2013

License: Freeware

Price: Free

Programming Language: Java

Features:

- Instant Run

- Intelligent code editor

- Robust and flexible build system

- Develop for all Android devices

- Performance profilers

Debugger:

Built-in Emulator: Yes (Fast and feature-rich)

Hardware Requirments: 3 GB of RAM, 2 GB of available disk space, and a screen resolution of 1280 by 800

Software Requirments: Microsoft Windows 7, 8, or 10 (32 or 64 bit)

Intergrated Development Environment (IDE): Android Studio does have an IDE and is where a developer could write the code for the application

Other Software Needed: The only thing that needs to be installed before creating apps (other then Android Studio Itself) is the Java Development Kit (JDK)

Advantages:

- Ability to refactor your code more effectively because Google has backed in a deeper support for specific android code.

- A polished User Interface Design that is fast and responds to changes more rapidly then other development tool software.

- The build tools for Android Studio allows for scripted builds which opens up many automation possibilities.

- Template-based wizards to create common Android designs and components

Dissadvantages:

- Higher hardware requirements compaired to other developer tool software

- Can only (Natively) support Java programming language.

Languages availability: English.

Uploading and Sell an Application: You would have to go through Google Play to Sell or Upload your application. Sadly you cant do it through Android Studio. So there is no licensing fee. But you are able to export the application through Android Studio.

Market Share: 88%

Ability to develop for wearable tech: Yes
