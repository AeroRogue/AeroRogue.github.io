Overview of monthly spending

tracks how much the user spends each month.

It shows the user how much he/she spends

during the month and compairing his/her

current spending to their average spending

on the current month. There is also an item

that shows the user where they are at with

there spending. Green = Good, Yellow = Average

Red = Overspending.

Notifies the user how much they spend and where (electronic recept)

When a user purchases something with their card it

notifies them on their cellphone. When they open the

application on their phone it shows them an

electronic recept. It shows the name of the business

and the amount spent.

Shows how much the user spends in a certain catagory

Shows the user how much they spend in certain

catagories. For example: If a user purchases

a video game it will show up in the entertainment

catagory. This is also recorded each month. For

example: Users can view how much they spend in

entertainment, utilities, etc. each month. Not

only that but it also notifies the user if they

spend to much in a certain catagory causing a

"Catagory-Spike".
